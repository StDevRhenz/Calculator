A feature-rich calculator app built with React Native and Expo, featuring multiple calculation modes and advanced mathematical functions.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start the App**
   ```bash
   npx expo start
   ```

3. **Run on Device/Simulator**
   - Press `a` for Android
   - Press `i` for iOS
   - Press `w` for web
